# rh124_x
* Extra Practice for RH124 Training - RHEL 8.2 Admin I
  >> All the script here can be only used on Redhat RH124 Lab Systems!

### Instruction 

1. clone this repo
```sh 
cd ~
git clone https://github.com/stv707/rh124_x.git
```

2. cd into ./rh124_x and run setup.sh 
```sh 
cd ./rh124_x
bash setup.sh 
```
